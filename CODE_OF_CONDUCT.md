# Code of Conduct

## Our Pledge
We are committed to making participation in this project a respectful,
harassment-free experience for everyone.

## Expected Behavior
- Be respectful and constructive.
- Assume good intent and provide actionable feedback.
- Focus on technical clarity and reproducibility.

## Unacceptable Behavior
- Harassment, personal attacks, discrimination, or abusive language.
- Deliberate misinformation in artifacts or validation records.

## Enforcement
Project maintainers may remove, edit, or reject contributions that violate
this code of conduct.
