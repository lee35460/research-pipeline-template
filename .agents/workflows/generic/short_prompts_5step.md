# Generic Short Prompts (5-Step)

1) Capture
"Summarize discussion into design_draft. Separate fact / interpretation / decision / open_question."

2) Structure
"Using approved NotebookLM sources, structure architecture / mathematical formulation / paper structure. Mark TBD."

3) Spec
"Convert approved design to implementation_spec with modules, signatures, shapes, equation-code mapping, constraints, validation criteria. Do not code yet."

4) Implement
"Generate code strictly from implementation_spec. Do not modify architecture or invent logic."

5) Validate
"Generate validation_report for source grounding, architecture consistency, math-code consistency, paper-code consistency, hidden assumptions, and TBD."
