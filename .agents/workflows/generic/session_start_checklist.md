# Generic Session Start Checklist

## 체크 항목
1. 현재 작업 단계 확인
- [ ] Knowledge Capture
- [ ] NotebookLM Storage
- [ ] Structuring
- [ ] Implementation Spec
- [ ] Code Implementation
- [ ] Final Validation

2. 근거 상태 확인
- [ ] Source grounding 확인
- [ ] approved 문서 존재 확인
- [ ] draft 오사용 없음

3. 구현 게이트
- [ ] implementation_spec 존재
- [ ] spec에 함수/shape/수식 대응 포함

4. 완료 게이트
- [ ] validation_report 준비
- [ ] pass/fail 기준 명확화
